## Запуск проекта

### Требования

- Node.js (версия 18 или выше)
- JDK 17 или выше
- PostgreSQL 16 с расширением PostGIS
- S3-совместимое хранилище (или Amazon S3)

### Backend

#### Windows

```bash
cd server
mvn clean package -DskipTests
cd ..
docker compose up --build
```

#### Linux / macOS

```bash
chmod +x rebuild-all.sh
RUN_DOCKER=true ./rebuild-all.sh
```

### Frontend
```bash
cd frontend
npm install
npm start
```

### Переменные окружения
Для корректной работы необходимо настроить:

- Подключение к PostgreSQL (URL, имя пользователя, пароль)

- Ключи доступа к S3

- Настройки почтового сервера для отправки 2FA-кодов

- API-ключ Yandex Map API
